LEADPULSE — швидкий старт до прибутку (сьогодні → завтра)
============================================================

ЩО ВЖЕ ПРАЦЮЄ
-------------
- Сайт збирає ліди у Google Sheet (лист Leads).
- У колонці lead_score сортуйте за спаданням — спочатку дзвоніть 80+.
- Колонка status: new → contacted → qualified → won / lost.

НАЛАШТУВАННЯ СПОВІЩЕНЬ (5 хв)
-----------------------------
У Apps Script: Project Settings → Script properties → додайте:
  ALERT_EMAIL = ваш@email.com
Потім Deploy → керування → нова версія Web App (як раніше).
На кожну заявку прийде лист.

CALENDLY (5 хв)
---------------
1. Зареєструйте безкоштовний https://calendly.com
2. Створіть тип події 15 хв.
3. Вставте посилання у файл config.js → CALENDLY_URL на диску D:\Projects\Get Money
4. Залийте оновлений config.js у репозиторій GitHub (або скажіть — оновлю через API).

ЩОДЕННИЙ РЕЖИМ (25–40 хв)
--------------------------
Ранок:
1) Відкрити Leads, фільтр status = new.
2) Топ-10 за lead_score — надіслати Email #1 (нижче).
3) Якщо є телефон — дзвінок або голосове WhatsApp (скрипт нижче).

Вечір:
4) Оновити status у таблиці.
5) Follow-up тим, хто не відповів (Email #2).
